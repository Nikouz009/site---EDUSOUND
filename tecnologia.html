ESP32 SoC

Placa microcontroladora equipada com Wi-Fi nativo e clock de 240MHz. Atua como o núcleo de processamento do sistema.

Sensor de Som

Microfone de eletreto analógico calibrado para captar a amplitude de onda e oscilações acústicas da sala continuamente.

Atuador visual responsável pelo feedback imediato (Semáforo Acústico), alternando de cor baseado nos decibéis computados.

// Firmware EduSound — ESP32 + Google Sheets
#include <WiFi.h>
#include <HTTPClient.h>

const int PIN_MICROFONE = 34;  // Entrada analógica
const int PIN_LED_VERDE = 25;  // Feedback Visual (Ideal)
const int PIN_LED_AMARELO = 26; // Feedback Visual (Atenção)
const int PIN_LED_VERMELHO = 27;// Feedback Visual (Excessivo)

void setup() {
  Serial.begin(115200);
  pinMode(PIN_MICROFONE, INPUT);
  pinMode(PIN_LED_VERDE, OUTPUT);
  pinMode(PIN_LED_AMARELO, OUTPUT);
  pinMode(PIN_LED_VERMELHO, OUTPUT);
}

void loop() {
  int ruidovolt = analogRead(PIN_MICROFONE);
  float dB = converteParaDecibeis(ruidovolt);
  if (dB <= 55) { digitalWrite(PIN_LED_VERDE, HIGH); }
  delay(2000);
}
